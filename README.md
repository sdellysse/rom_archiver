To install dependencies:

```sh
bun install
```

To run:

```sh
bun run dev
```

open http://localhost:3000

Notes:

root dir of roms should look like this:

```
ROOT
  - snes
    - bios.console
    - bios.accessories
    - games
      - Legend of Zelda - A Link to the Past, The
        - base_roms
          - r_usa__v_1.0
        - patches
          - msu1
        - msu1_soundtracks
```
